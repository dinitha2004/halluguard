# HalluGuard Project Structure

## app
Frontend and application flow.

## data/raw
Original source datasets.

## data/processed
Cleaned and transformed datasets.

## data/labeled
Token-level and span-level labeled data.

## data/features
Extracted hidden-state features and training-ready arrays.

## data/splits
Train, validation, and test split files.

## db
SQLite database logic and local database file.

## models
Core model logic and saved lightweight probe models.

## outputs
Experiment outputs, predictions, logs, and metric files.

## reports/figures
Charts, diagrams, and images used in the report and presentation.

## training
Scripts for dataset building, probe training, and evaluation.

## configs
Reusable configuration files for experiments and thresholds.
